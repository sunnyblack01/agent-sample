# 화학실험 프로그램 면접 준비 가이드

---

## 1. 보고서 기반 예상 질문

### Q1. "시계반응이 뭔지 간단히 설명해볼래요?"
- **의도:** 보고서 본인 작성 여부 확인
- **핵심:** 반응이 보이지 않다가 특정 시점에 갑자기 색 변화, "시계"라는 이름 = 시간 측정에 유리
- **주의:** 반응식 암기 줄줄 읊지 말고, 왜 이 반응을 선택했는지와 연결

### Q2. "18가지 조건을 왜 그렇게 나눴어요?"
- **의도:** 실험 설계 능력 확인
- **핵심:** 3×3×2=18, 3단계인 이유 = 2개면 직선밖에 못 그림, 3개면 곡선 경향 파악 가능
- **주의:** "교과서에서 배워서"가 아니라 논리적 이유

### Q3. "14개 학습, 4개 검증은 어떤 기준?"
- **의도:** Train/Test 개념 진짜 이해하는지
- **핵심:** 테스트 4개가 온도·농도·촉매 골고루 분포되게 선택
- **주의:** "머신러닝에서 쓰는 방법"만 말하면 안 됨, 왜 그러는지 본인 말로

### Q4. "3가지 예측 방법 중 어떤 게 가장 정확했어요?"
- **의도:** 결과 비교/해석 능력
- **핵심:** 오차율 수치 기억 + 왜 그 방법이 더 좋았는지 본인 해석
- **주의:** 숫자 정확히 모르면 "대략" 쓰되, 경향은 반드시 기억

### Q5. "실험에서 가장 어려웠던 점은?"
- **의도:** 실제 실험 경험 확인
- **핵심:** 구체적 어려움 1가지 + 어떻게 해결/보완했는지
- **주의:** "딱히 없었다"는 최악의 답변

### Q6. "예측값과 실제값 차이 이유는?"
- **의도:** 오차 분석 능력
- **핵심:** 실험적 오차(측정)와 모델 한계(가정)를 구분해서 말하기

### Q7. "다시 한다면 뭘 바꾸고 싶어요?"
- **의도:** 개선 의지, 비판적 사고
- **핵심:** 구체적 한계 → 구체적 개선 순서로

## 2. 진로 연결 질문

### Q1. "왜 시계반응에 예측 모델을?"
- **프레임:** 화학 수업에서 패턴 느낌 → 예측해보고 싶었음 → AI 기본 원리와 같은 구조
- **에피소드:** 아레니우스 식 배울 때 "이걸 데이터로 직접 확인하면?" 생각한 순간

### Q2. "AI 학과 가고 싶으면서 왜 화학실험 프로그램?"
- **핵심:** AI는 도구, 적용할 대상이 필요. "AI만 아는 사람"보다 "AI를 적용할 분야를 아는 사람"
- **에피소드:** 데이터 정리하면서 "데이터의 질이 예측을 좌우한다" 느낀 경험

### Q3. "AI로 구체적으로 뭘 하고 싶어요?"
- **핵심:** 과학 데이터 분석/예측 분야 관심. 아직 확정은 아니지만 방향 잡는 중
- **에피소드:** 물리 역학 + 이번 화학 실험 = 다양한 분야 데이터에 AI 적용 가능성

### Q4. "이 예측 방법이랑 실제 AI 기술이랑 뭐가 달라요?"
- **핵심:** 이번엔 사람이 규칙 정함 vs AI는 스스로 찾음. 공통점은 데이터→패턴→예측→검증 흐름

## 3. 돌발/심화 질문

### Q1. "아레니우스 식 알아요?"
- 알면 간단히 설명, 모르면 "이름은 들어봤지만 수식까지는 아직"
- "이 프로그램을 통해 더 배우고 싶습니다"로 마무리

### Q2. "촉매가 반응 속도를 빠르게 하는 원리는?"
- 알면 "활성화 에너지를 낮춘다" + 실험에서 확인한 데이터 연결
- 모르면 "원리는 아직 깊이 몰라도 실험 결과로 확인했다"

### Q3. "곱셈 모델에서 변인 간 상호작용이 있으면?"
- "각 변인이 독립적이라고 가정했는데, 실제로는 상호작용이 있을 수 있다"
- "그래서 특정 조건에서 오차가 더 컸을 수 있다"

### Q4. "머신러닝이랑 딥러닝 차이는?"
- 완벽한 정의보다 본인 수준에서 솔직하게
- "직접 구현해본 적은 없지만 이번에 기본 원리를 체감했다"

### Q5. "반복 실험은 했어요?"
- 했으면: "각 조건당 3회 반복, 평균값 사용"
- 안 했으면: 솔직히 + "한계로 인식하고 있다"

## 4. 면접 시뮬레이션 (5분)

| 시간 | 질문 | 핵심 포인트 |
|------|------|-------------|
| 0:00~0:30 | 자기소개 + 지원 동기 | 이름, 학년, AI 진로, "화학 데이터를 직접 다뤄보고 싶었다" |
| 0:30~2:00 | 14개 학습/4개 검증 방법과 결과 | 큰 그림 → 가장 정확한 방법 → 오차 원인 |
| 2:00~3:30 | AI와의 연결 | 데이터→패턴→예측→검증 = AI 기본 흐름, 차이점도 아는 것 |
| 3:30~5:00 | 부족한 점 + 배우고 싶은 것 | 솔직한 한계 1가지 + "부족한 부분을 채우러 왔다" |

| 패턴 | 왜 위험 | 대안 |
|------|---------|------|
| "ChatGPT로 분석" | 도구 의존 | 본인이 직접 한 과정 중심 |
| 교과서 문장 암기 | 이해 없이 외운 느낌 | 본인 말 + 비유 |

| "완벽하게 했다" | 비판적 사고 부재 | 한계 아는 게 더 높은 평가 |
| 2분 이상 답변 | 집중력 저하 | 핵심→근거→마무리, 1분 내외 |

## 준비 체크리스트

- [ ] 보고서 핵심 숫자(조건 수, 오차율) 기억
- [ ] 3가지 예측 방법 차이를 30초 안에 설명 연습
- [ ] "모릅니다. 하지만 ~"으로 이어가는 답변 연습
- [ ] 시뮬레이션 최소 2회 연습 (녹음/거울)
- [ ] 면접 당일 보고서 사본 지참
