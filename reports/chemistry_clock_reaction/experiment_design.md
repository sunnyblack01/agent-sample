# 시계반응을 이용한 반응 속도 예측 모델 구축 — 실험 설계서

---

## 1. 탐구 질문

**"아이오딘 시계반응에서 온도, 반응물 농도, 촉매 유무를 변화시키면 반응 시간(색 변화까지 걸리는 시간)은 어떻게 달라지며, 일부 조건의 실험 결과만으로 나머지 조건의 반응 시간을 정확히 예측할 수 있을까?"**

---

## 2. 가설

| 변인 | 가설 | 이론적 근거 |
|------|------|-------------|
| **온도** | 온도가 높을수록 반응 시간이 짧아질 것이다 | 아레니우스 식에 따르면 온도 10°C 상승 시 반응 속도 약 2~3배 증가 |
| **농도** | KIO₃ 농도가 높을수록 반응 시간이 짧아질 것이다 | 반응 속도 법칙(v = k[A]^m[B]^n), 농도↑ → 충돌 횟수↑ |
| **촉매** | 촉매 첨가 시 반응 시간이 짧아질 것이다 | 촉매는 활성화 에너지를 낮춰 반응 속도를 높임 |
| **예측** | 14개 조건 데이터로 나머지 4개 조건을 ±20% 이내로 예측 가능할 것이다 | 온도·농도·촉매의 영향이 체계적 패턴을 따르므로 외삽/비례식 가능 |

---

## 3. 변인 설정

### 독립변인

| 독립변인 | 단계 | 구체적 수치 |
|----------|------|-------------|
| 온도 | 3단계 | 10°C (얼음물 수조), 25°C (실온), 40°C (온수 수조) |
| KIO₃ 농도 | 3단계 | 0.02 M, 0.04 M, 0.08 M (2배씩 증가) |
| 촉매 유무 | 2단계 | 무촉매 / 몰리브덴산 암모늄 0.1% 수용액 2mL |

> 촉매 대체재: 황산망간(II) MnSO₄ 0.1M 수용액 2mL

### 종속변인

| 종속변인 | 측정 방법 |
|----------|-----------|
| 반응 시간(초) | 혼합 시점 ~ 진한 남색 변화 시점, 흰 종이 × 표시 소멸 기준 |

### 통제변인

| 통제변인 | 통제 방법 |
|----------|-----------|
| NaHSO₃ 농도 | 모든 조건 0.01 M 고정 |
| 전분 지시약 | 1% 전분 용액 1 mL |
| 용액 총 부피 | 혼합 후 50 mL (증류수로 보정) |
| 혼합 방법 | 동일인, 한 번에 붓고 유리 막대 3회 교반 |
| 종점 판정 | 흰 종이 × 표시 소멸 시점 |
| 온도 평형 시간 | 혼합 전 5분간 수조에서 평형 |

## 5. 실험 절차

### 사전 준비: 시약 조제

**① KIO₃ 모액 0.08 M, 500 mL**
- KIO₃ 8.56 g → 500 mL 부피 플라스크, 증류수로 표선
- 희석: 0.04 M = 모액 250 mL + 증류수 250 mL
- 희석: 0.02 M = 모액 125 mL + 증류수 375 mL

**② NaHSO₃ 0.01 M, 500 mL**
- NaHSO₃ 0.52 g → 500 mL 부피 플라스크
- ⚠ 실험 당일 제조, 밀폐 보관

**③ 1% 전분 지시약 200 mL**
- 전분 2 g을 냉수 20 mL에 현탁 → 끓는 물 180 mL에 넣어 호화 → 냉각

**④ 촉매 용액 0.1% (NH₄)₂MoO₄ 200 mL**
- 0.2 g + 증류수 200 mL

### 본 실험

**1단계: 수조 세팅**
- 10°C: 얼음물 / 25°C: 실온 / 40°C: 온수

**2단계: 용액 준비 (1회분)**

| 성분 | 비커 | 양 |
|------|------|-----|
| KIO₃ 용액 | 비커 A | 20 mL |
| 전분 지시약 | 비커 A | 1 mL |
| 촉매 (또는 증류수) | 비커 A | 2 mL |
| NaHSO₃ | 비커 B | 20 mL |
| 증류수 | 보정용 | 총 50 mL 맞춤 |

→ 비커 A, B를 수조에서 5분간 온도 평형

**3단계: 반응 및 측정**
1. 흰 종이 위 × 표시, 그 위에 100 mL 비커
2. 비커 A 내용물을 먼저 부음
3. 비커 B를 한 번에 부으며 스톱워치 시작
4. 유리 막대 3회 교반 후 관찰
5. 남색 변화 → × 소멸 시점에 스톱워치 정지, 기록
6. 동일 조건 3회 반복

**실험 순서**: 25°C(6가지) → 40°C(6가지) → 10°C(6가지)

### 안전 주의사항
- 보안경·실험복·장갑 착용
- KIO₃ 피부 접촉 시 즉시 세척
- NaHSO₃ 사용 시 환기 필수
- 가열판 화상 주의
- 폐액 지정 용기에 수거

## 7. 예측-검증 설계 (핵심!)

### 7-1. 검증용 4가지 조건

| 역할 | 조건 | 온도 | 농도 | 촉매 | 선택 이유 |
|------|------|------|------|------|-----------|
| 검증 1 | C04 | 10°C | 0.04 M | 유 | 저온, 중간농도, 촉매유 |
| 검증 2 | C08 | 25°C | 0.02 M | 유 | 중온, 저농도, 촉매유 |
| 검증 3 | C11 | 25°C | 0.08 M | 무 | 중온, 고농도, 촉매무 |
| 검증 4 | C15 | 40°C | 0.04 M | 무 | 고온, 중간농도, 촉매무 |

### 7-2. 예측 방법 3가지

**방법 A: 그래프 외삽/내삽법**
- 농도-반응시간, 온도-반응시간 그래프에서 빠진 점의 Y값 읽기

**방법 B: 비례식(배율법)**
- 촉매 효과 비율, 농도 효과 비율을 다른 조건에서 계산 → 적용

**방법 C: 곱셈 모델**
- t = 기본값 × f(온도) × g(농도) × h(촉매)
- 14개 데이터로 각 계수 산출 → 예측

## 8. AI 관점 연결

### 구조적 대응

| 이 실험 | AI/머신러닝 | 설명 |
|---------|------------|------|
| 18가지 조건 | 전체 데이터셋 | 알고 싶은 모든 경우의 수 |
| 온도, 농도, 촉매 | 입력 특성(Features) | 모델에 넣는 입력값 |
| 반응 시간 | 출력(Label) | 예측하려는 값 |
| 14개 학습 데이터 | 훈련 세트(Training Set) | 패턴을 배우는 데이터 |
| 4개 검증 데이터 | 테스트 세트(Test Set) | 예측 정확도 평가용 |
| 비례식·외삽 | 모델(Model) | 패턴을 수식으로 표현 |
| 오차율 비교 | 모델 평가(Evaluation) | 예측 정확도 측정 |
| 3가지 방법 비교 | 모델 선택(Model Selection) | 최적 모델 고르기 |

### 더 탐구할 수 있는 질문
- "14개 대신 9개로 학습하면 정확도가 떨어질까?" → 데이터 양과 성능
- "검증 4개를 바꾸면?" → 테스트 세트 구성의 중요성
- "더 복잡한 모델을 쓰면?" → 과적합(Overfitting) 개념

## 실험 흐름 요약

```
시약 조제 → 수조 세팅 → 14개 학습 조건 실험(각 3회)
→ 그래프 + 비례식 + 곱셈 모델로 패턴 파악
→ 4개 검증 조건 반응 시간 "예측"
→ 4개 검증 조건 실제 실험(각 3회)
→ 예측값 vs 실측값 비교 → 오차율 분석
→ 어떤 예측 방법이 가장 정확했는지 결론
→ AI Train/Test Split과의 연결 고찰
```

총 18조건 × 3회 = 54회 측정, 약 3~4시간 소요
