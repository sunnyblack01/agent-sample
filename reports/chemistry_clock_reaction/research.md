## 1. 교과서 기반 이론 정리

### 1-1. 아이오딘 시계반응의 원리

아이오딘 시계반응(Landolt reaction)은 1886년 스위스 화학자 란돌트가 관찰한 반응으로, 두 가지 무색 용액을 혼합한 뒤 일정 시간이 지나면 갑자기 짙은 청남색으로 변하는 반응이다.

**반응 메커니즘 (2단계):**
- **1단계 (느린 반응):** KIO₃가 NaHSO₃에 의해 환원되어 I₂ 생성
- **2단계 (빠른 반응):** 생성된 I₂가 남은 NaHSO₃에 의해 다시 I⁻로 환원. NaHSO₃가 모두 소모되면 I₂가 축적 → 전분과 반응 → 청남색

**핵심:** 색 변화까지의 유도 시간 측정으로 반응 속도를 정량 비교 가능

### 1-2. 반응 속도 영향 요인

| 요인 | 원리 | 시계반응에서의 관찰 |
|------|------|---------------------|
| 농도 | 농도↑ → 충돌 빈도↑ → 속도↑ | KIO₃ 농도↑ → 유도 시간↓ |
| 온도 | 온도↑ → 운동에너지↑ → 활성화 에너지 이상 분자 비율↑ | 수조 온도↑ → 유도 시간↓ |
| 촉매 | 활성화 에너지↓ → 반응 경로 변경 | 촉매 첨가 → 유도 시간↓ |

### 1-3. 아레니우스 식 (고등학생 수준)

**k = A × e^(-Ea/RT)**

- k: 반응속도상수 (클수록 반응 빠름)
- A: 빈도인자
- Ea: 활성화 에너지
- R: 기체 상수 (8.314 J/mol·K)
- T: 절대 온도 (K)

해석:
- 온도(T)↑ → k↑ → 반응 빨라짐
- 촉매 → Ea↓ → k↑ → 반응 빨라짐
- ln(k) vs 1/T 그래프 = 직선 → 기울기로 활성화 에너지 산출 가능
